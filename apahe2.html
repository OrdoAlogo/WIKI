Teniendo el ubuntu server instalado o cualquier otro sistema linux podemos instalar un servicio Apache para acceder a la pagina web
Lo primero que tendremos que hacer ser√° instalar el servicio:

sudo apt install apache2

una vez instalado el apache iremos a la ruta /etc/apache2/sites-available con el siguiente comando

cd /etc/apache2/sites-available

una vez en este sitio crearemos un archivo utilizando el nombre de nuestro dominio. El nuestro sera dominio.

sudo nano dominio.com.conf

Este archivo nos aparecera vacio y tendremos que configurarlo de la siguiente manera

< VirtualHost *:80>
ServerAdmin admin@dominio.com
ServerName dominio.com
ServerName dominio.com
ServerAlias www.dominio.com
DocumentRoot /var/www/dominio.com/html # Aqui ira la ruta en la que pondremos nuestra pagina web, tenerd en cuenta que tendra que tener permisos 777
ErrorLog ${APACHE_LOG_DIR}/error.log
CustomLog ${APACHE_LOG_DIR}/access.log combined
< /VirtualHost>

Una vez configurado este archivo tendremos que habilitarlo para que el apache lo muestre al llamar al servidor para esto, ejecutaremos el siguiente comando

sudo a2ensite dominio.com.conf

El siguiente paso sera desabilitar el archivo que viene por defecto

sudo a2dissite 000-default.conf

Una vez echos estos pasos comprobaremos que todo esta bien con el siguiente comando. Cuando pongamos el comando nos saldra un OK

sudo apache2ctl configtest

por ultimo reiniciaremos el servicio y la pagina se vera cuando solicitemos www.dominio.com

sudo systemctl restart apache2

Felicidades ya has configurado tu apache2
