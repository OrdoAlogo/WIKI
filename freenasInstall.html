Instalación y configuración basica de FreeNAS.

Para realizar este tutorial partimos de una maquina virtual creada. Puede ver el tutorial aquí

1- Seleccionamos Install/Upgrade.

2- Elegimos el disco en el que freenas se instalará.

3- Seleccionamos Yes.

4- Una vez instalado reiniciamos el sistema.

5- Una vez reiniciado aparecerá el siguiente menú.

Al igual que en proxmox, este mostrará una ip para poder conectarse remotamente.

6- Si queremos configurar la ip y la máscara seleccionaremos la opción 1 y seguiremos los siguientes pasos.

7- Si queremos configurar la DNS seleccionaremos la opción 6 e introduciremos la DNS.
