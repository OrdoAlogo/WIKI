Instalación y configuración de FTP.

Teniendo el ubuntu server instalado o cualquier otro sistema linux podemos instalar un servicio FTP para acceder y almacenar información desde otro ordenador
Lo primero que tendremos que hacer será instalar el servicio:

sudo apt-get install vsftpd

Después de instalarlo tendremos que irnos al fichero de configuración y cambiar las siguientes líneas: Para acceder al fichero de configuración:

sudo nano /etc/vsftpd.conf

Tendremos que descomentar las siguientes líneas.

write_enable=YES
local_umask=022
chroot_local_user=YES

Una vez descomentadas esas lineas tendremos que escribir las siguientes lineas

allow_writeable_chroot=YES
pasv_enable=Yes
pasv_min_port=40000
pasv_max_port=40100

Una vez realizados estos pasos tendremos que reiniciar el servicio

sudo service vsftpd restart

Para finalizar comprobaremos que ese servicio está activo y funcionando

systemctl status vsftpd
